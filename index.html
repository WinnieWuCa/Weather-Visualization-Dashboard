Web Visualization Dashboard (Latitude)

Welcome to the Web Visualization Dashboard. The purpose of this project is to analyze how weather changes as you get closer to the the equator. We pulled data from the OpenWeatherMap API to assemble a dataset of > 500 cities.

After assembling the data, we used Matplotib to plot various factors of the weather vs. latitude, factors include cloudiness, humidity, temperature and windiness.
